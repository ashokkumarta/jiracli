# JIRA Command Line Interface

## Usage

    $ java -jar jiracli.jar -c
    Root URL: https://jira.atlassian.com/
    Username:
    jiracli> read | print ${summary}/${status.name}
    JRA-123
    New Report: Voted Issues/Closed
    
    jiracli> exit
